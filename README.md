# ⛏ Jogo do Mineiro — 2.5D

O **Jogo do Mineiro** é uma experiência de plataforma e exploração subterrânea com elementos de sobrevivência e gerenciamento de recursos. O jogador deve escavar galerias procedurais em busca de minérios valiosos, utilizando as riquezas encontradas para fabricar ferramentas avançadas e sobreviver às profundezas.

## 🚀 Funcionalidades e Mecânicas

- **Mineração e Economia:** Colete quatro tipos de recursos com valores diferentes:
  - ⬛ **Carvão:** Básico para tochas e energia.
  - ⬛ **Ferro:** Essencial para ferramentas estruturais.
  - ⬛ **Ouro:** Utilizado em explosivos e kits médicos.
  - ⬛ **Diamante:** Raro, necessário para tecnologias de ponta (Laser e Teleporte).
- **Sistema de Crafting e Ferramentas (Hotkeys 1-9):**
  - **🧨 Dinamite & 💣 Granada:** Destruição de blocos em massa e controle de área.
  - **🫧 Gás Extrator & 🚀 Foguete:** Mobilidade avançada e aumento de velocidade.
  - **☄️ Laser Minerador:** Abre túneis longos e precisos instantaneamente.
  - **🌀 Teleporte:** Salto dimensional para a próxima galeria.
- **Power-ups Dinâmicos:** Encontre itens no cenário que concedem **Super Velocidade (S)**, **Pulo Duplo (J)**, **Escudo (E)** e **Ímã de Minérios (M)**.
- **Ambiente 2.5D:** Gráficos que misturam a jogabilidade clássica lateral com profundidade visual e efeitos de partículas.

## 🛠️ Tecnologias Utilizadas

- **HTML5 Canvas:** Renderização do mundo procedural e sistema de partículas de iluminação.
- **Vanilla JavaScript:** Motor de física, sistema de inventário/crafting e geração de cavernas.
- **CSS3:** HUD (Heads-Up Display) estilizado para gerenciamento de ferramentas e recursos.

## 🕹️ Como Jogar

1. Acesse o [Jogo do Mineiro](https://jjunninho.github.io/jogo_mineiro/).
2. **Movimentação:** Use `A / D` para mover e `W / Espaço` para pular.
3. **Mineração:** Pressione `Z` ou `X` para usar a picareta básica.
4. **Ferramentas:** Use as teclas numéricas `1 a 9` para selecionar e ativar ferramentas criadas.
5. **Inventário:** Pressione `Q` para abrir o menu de ferramentas e ver os custos de criação.
6. **Objetivo:** Acumule pontos coletando diamantes e chegue o mais longe possível nas galerias.
